# Spring MVC 支持

`Restlight`支持了Spring MVC的使用习惯，你可以按照Spring MVC的方式使用Restlight 

引入依赖

```xml
<dependency>
	<groupId>io.esastack</groupId>
	<artifactId>restlight-springmvc-provider</artifactId>
	<version>${restlight.version}</version>
</dependency>
```
