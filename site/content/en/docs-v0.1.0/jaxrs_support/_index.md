    `Restlight`支持了JAX-RS的使用习惯，你可以按照JAX-RS的方式使用Restlight

引入依赖

```xml
<dependency>
	<groupId>io.esastack</groupId>
	<artifactId>restlight-jaxrs-provider</artifactId>
	<version>${restlight.version}</version>
</dependency>
```
